Comment se connecter ?

Pour vous connecter à l'application vous devez entrer le nom de votre entreprise, ainsi que le mot de passe qui vous a été donné.

Cliquer ensuite sur "se connecter", si les informations précédentes sont incorectes un message d'erreur aparaîtra.

Comment gérer les sondages ?

Création :

Pour créer un sondage, cliquer sur le bouton "Ajouter" lorsque vous vous trouvez sur la page de gestion des sondages.

Une nouvelle page aparaîtra dans laquelle vous pouvez nommer votre sondage ainsi que gérer les questions associées à celui-ci.


Modification :

Pour modifer un sondage, seléctionner le sondage voulu et cliquer sur le bouton "Modifier" lorsque vous vous trouvez sur la page de gestion des sondages.

Une nouvelle page aparaîtra dans laquelle vous pouvez renommer votre sondage ainsi que gérer les questions associées à celui-ci.


Suppression :

Pour supprimer un sondage, seléctionner le sondage voulu cliquer sur le bouton "Supprimer" lorsque vous vous trouvez sur la page de gestion des sondages.

Un message vous demandera si vous êtes sûr de vouloir supprimer ce sondage. Une réponse positive supprimera le sondage.

Attention, la suppression d'un sondage est iréversible.

Comment gérer les questions d'un sondage ?

Création :

Pour créer une nouvelle question, cliquer sur le bouton "Ajouter".

Une nouvelle fenêtre s'ouvrira dans laquelle vous pouvez intituler votre question ainsi que gérer les réponses associées à celui-ci.

L'ajout de réponse se fait en cliquant sur le bouton "Ajouter" de la nouvelle fenêtre. Vous pouvez renommer les réponses en double cliquant dessus.

Pour supprimer une réponse, selectionné là et cliquer sur "Supprimer".

Si votre question peut avoir plusieurs réponses, cocher "Choix multiples".

Si votre question doit être répondu à l'aide d'une valeur numérique, cocher "Réponses numériques".


Modification :

Pour modifer une question, selectionner la question voulue et cliquer sur le bouton "Ajouter".

Une nouvelle fenêtre s'ouvrira dans laquelle vous pouvez modifier votre question ainsi que gérer les réponses associées à celui-ci.

L'ajout de réponse se fait en cliquant sur le bouton "Ajouter" de la nouvelle fenêtre. Vous pouvez renommer les réponses en double cliquant dessus.

Pour supprimer une réponse, selectionné là et cliquer sur "Supprimer".

Si votre question peut avoir plusieurs réponses, cocher "Choix multiples".

Si votre question doit être répondu à l'aide d'une valeur numérique, cocher "Réponses numériques".


Suppression :

Pour supprimer une question, seléctionner la question voulue et cliquer sur le bouton "Supprimer" lorsque vous vous trouvez sur la page de gestion des questions.

Un message vous demandera si vous êtes sûr de vouloir supprimer cette question. Une réponse positive supprimera la question.

Attention, la suppression d'une question est iréversible.
